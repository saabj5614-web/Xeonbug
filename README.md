## ✨ Key Features

- Multi WhatsApp sessions via pairing code
- Telegram bot dashboard to manage everything
- Many working crash / bug methods (Android + iOS + Groups)
- Premium / Reseller system with access control
- Force join channel & group protection
- Auto-reconnect sessions
- Custom branding per bot instance
- Broadcast messages to all users
- List all WhatsApp groups of connected numbers

<br>

## 🚀 Installation & Setup

```bash
# 1. Clone the repo
git clone https://github.com/SABIR7718/XeonTGbot.git
cd XeonTGbot

# 2. Install dependencies
npm install

# 3. Configure (very important!)
cp config.example.js config.js   # if you have example
# or directly edit config.js

# 4. Edit config.js (main settings)
nano config.js

# 5. Start the bot
npm start
# or
node index.js
```
